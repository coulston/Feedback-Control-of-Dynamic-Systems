\begin{enumerate}[(a)]
\item  Find the transfer function for the tank from input $q_{in}$ to output $h$. 
\item Draw a block diagram that represents the PI feedback control system. Include the reference signal for the desired height $H_{d}(s)$
\item  Determine the acceptable damping ratio and undamped natural frequency of the closed loop poles achieve the desired closed loop specifications
\item Find the gains for your controller that achieve the pole locations for part (c)
\end{enumerate}
