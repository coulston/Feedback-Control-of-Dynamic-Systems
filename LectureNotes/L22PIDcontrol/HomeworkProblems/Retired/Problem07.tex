The oven has a thermal capacitance of $100$ J/K, and the thermal resistance to the ambient temperature $T_{a}$ is $25$  K/W. 
\begin{enumerate}[(a)]
\item Find the transfer function for the oven from input $\dot{Q}_{in}$ to output $T_{o}$. (We will ignore variations in $T_{a}$ for this problem.) 
\item Draw a block diagram that represents the PI feedback control system. Include the reference signal for the desired temperature $T_{d}(s)$.
\item Determine the acceptable damping ratio and undamped natural frequency of the closed loop poles to achieve the desired closed loop specifications. Assume no effect due to zeros.
\item Find the gains for your controller that achieve the pole locations for part (c).
\item Implement your feedback system in Simulink, following the example in the PID Control lecture. Plot the response to a reference step input.
