\begin{itemize}
\item Settling time equal to 10 seconds
\item Steady state error for unit ramp input equal to 0.1 degrees
\end{itemize}\vspace{.05in}

The oven has a thermal capacitance of $10$ J/K, and the thermal resistance to the outside is $44$  (m$^2$K)/W. 
\begin{enumerate}
\item Find the transfer function for the oven from input $\dot{Q}_{in}$ to output $T_{o}$. (We will ignore variations in $T_{a}$ for this problem) 
\item Draw a block diagram that represents the PI feedback control system. Include the reference signal for the desired temperature $T_{d}(s)$
\item Find the gains for your controller that achieve the desired closed loop specifications
\item Implement your feedback system directly in Matlab or using Simulink.  Plot the response to a reference step input. %For Simulink help, follow the example in Sections 5 and 6 of Lecture \PIDControlNumber (\PIDControlName).
