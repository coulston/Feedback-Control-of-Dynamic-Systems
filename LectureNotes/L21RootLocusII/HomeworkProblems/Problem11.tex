A student designs a PD controller $C(s) = K_{p}+K_{d}s$ for the system $G(s) = \frac{1}{s(s+2)}$, using a unity gain feedback configuration. Use root locus arguments to determine a value for the ratio $K_{p}/K_{d}$ that will result in a stable, overdamped response for all values of $K_{d}$  (e.g., all poles on the real axis in the open LHP).
