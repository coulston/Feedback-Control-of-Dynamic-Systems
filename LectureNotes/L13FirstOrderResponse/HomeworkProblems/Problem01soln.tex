The system is first order with pole magnitude $\sigma = 6$. The settling time is $\ts = \frac{4.6}{\sigma} = \frac{4.6}{6} = 0.77$ s. The rise time is $\tr = \frac{2.2}{\sigma} = \frac{2.2}{6} = 0.37$ s.
