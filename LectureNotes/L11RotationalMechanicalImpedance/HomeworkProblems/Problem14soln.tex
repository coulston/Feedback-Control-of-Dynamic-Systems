\end{align*}
Using residue theorem,
\begin{align*}
A &= \left.  \frac{1}{(s+1)(s+2)} \right|_{s=0} = 0.5
B &= \left.  \frac{1}{s(s+2)} \right|_{s=-1} = -1
C &= \left.  \frac{1}{s(s+1)} \right|_{s=-2} = 0.5
\end{align*}
Thus
\[
Y_{out}(s) = \frac{1}{s}  - \frac{1}{(s+1)} + \frac{-1}{(s+2)}
