Determine if the following systems can be approximated by either a 1st or 2nd order system without zeros, and if so, find the approximation
\begin{enumerate}
\item[(a)] $G(s) =\frac{(s+5)}{(s+.1)(s^{2}+2s+2)}$
\item[(b)] $G(s) =\frac{(s^2+2s+2)}{(s+1)(s^2+5s+10)}$
\item[(c)] $G(s) =\frac{(s^2+2s+2)}{(s+1)(s^2+20s+100)}$
\end{enumerate}
